Условия возврата

Возврат товара надлежащего качества:

  • Товар не подошел по размеру, цвету, форме
  • Товар не подошел по габаритам
  • Товар не подошел по комплектации

Возврат товара ненадлежащего качества:

  • Обнаружен производственный брак
  • Товар не соответствует описанию
  • Товар не работает или работает некорректно

Важная информация

Не подлежат возврату товары надлежащего качества:

  • Товары с нарушенной упаковкой
  • Товары с нарушенными пломбами
  • Товары с нарушенной комплектацией
  • Товары с признаками использования
